# v15.1.0 small-scale adaptive network MVP

## objective

Validate that a small-scale Gossamer-Link network can:

- maintain metastable structural adaptation
- react differently to multiple stimuli
- preserve diversity under weak human pressure
- avoid trust saturation and dominance collapse

---

## key implementation points

### deterministic reproducibility

- fixed random seed
- fixed node count
- fixed trust boundaries
- fixed variance limits

### adaptive structure

- trust-based connection rewiring
- diversity-preserving essence updates
- weak external human influence only
- anomaly-aware trust stabilization

### metastability safeguards

- lower variance recovery
- upper variance clamp
- anti-dominance trust ranking
- diversity repulsion mechanism

---

## validation criteria

### required conditions

- mean_variance:
  0.02 <= variance <= 0.06

- trust_range:
  trust_range >= 0.2

- structural_diversity:
  diversity > 0.08

- dominance_ratio:
  dominance_ratio < 0.5

- trust_saturation:
  must remain False

---
